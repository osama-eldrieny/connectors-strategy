Nintex · Connector Strategy
in progress

Connector Acceleration

Using User Flow to get feedback from users on missing connectors, missing actions, finding connections, creation ease, and awareness

🎯 Research Priorities

PRIMARY (70%)

  • Missing connectors (which ones, why)
  • Missing actions in existing connectors
  • Business criticality/importance
  • Impact & timeline

SECONDARY (30%)

  • Ease of creating connections
  • Ease of finding connectors
  • Awareness of unified behavior

👥 Journey 2: Existing User (Connection Creation Feedback) Primary 70%

9-step feedback survey with score-based branching on creation and discovery ease

Step 1: How easy was it to create this connection? → Step 2: [Hidden - Branching: if ≤3 go to Step 3 | if 4-6 go to Step 3 | if ≥7 go to Step 6] → Step 3: What was difficult about creating this connection? → Step 4: [Hidden - Branching: if ≤3 go to Step 5 | if 4-6 go to Step 6] → Step 5: Can we schedule a 15-minute call to learn more about your integration needs? → Step 6: How easy was it to find the connector you needed? → Step 7: [Hidden - Branching: if ≤6 go to Step 8 | if ≥7 go to Step 9] → Step 8: What would help you find connectors more easily? → Step 9: Thank you!
Step 1

How easy was it to create this connection?

Creation ease rating with score-based branching

Trigger
User clicks "Done" button in permissions dialog (3-second delay, once per user)
Type
Rating Scale (1-10)
1–10 numeric scale with NO custom anchor labels
Lead-in Text
🎉 Nice work! You've successfully created a connection.
Saved to Attribute
"Create Connection Scale"
Branching
If ≤3: Go to Step 3 → Step 5 → Step 9 | If 4-6: Go to Step 3 → Step 6 | If ≥7: Skip to Step 6
